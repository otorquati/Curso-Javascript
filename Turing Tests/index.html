Baseball Game

Este é um programa que analisa uma cadeia de dados passados separados por vígurla conforme abaixo:

  • Caso o dado seja um número este é anotado ao placar.
  • Caso seja "+" o dado é somado ao placar;
  • Caso seja "D" anota o dado em dobro no placar;
  • Caso seja "C" invalida o dado previamente anotado no placar;
